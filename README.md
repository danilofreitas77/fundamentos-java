# Fundamentos Java - Lógica, Condicionais e Repetição

Repositório com exercícios práticos desenvolvidos durante meus estudos de Java, com foco em:

- Lógica de Programaçao;
- Estruturas condicionais (if, else, else if e switch);
- Estruturas de repetição;
- Entrada de dados com Scanner;
- Operações matemáticas e manipulação de variáveis

A ideia aqui não é só “resolver exercício”, mas construir base sólida de raciocínio lógico.

## Tecnologias utilizadas

- Java;
- IntelliJ IDEA;
- Git & GitHub

## Estutura do projeto

O projeto está estruturado em duas pastas (lógica / condicionais_repeticao) com exercicios dentro de cada respectiva pasta

## Objetivo

Fortalecer fundamentos da programação para evoluir para:

- Programação orientada a objetos
- Desenvolvimento backend
- Estruturas de dados
- Projetos mais robustos

Base forte = evolução rápida.
